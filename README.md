# NFC Reader App

A simple Android application to read NFC tags and display their data.

## Features
- Detects NFC tags (NDEF, Tech, Tag discovered).
- Reads Tag ID.
- Parses NDEF text records.
- Displays data on screen.

## Setup
1. Open this project in **Android Studio**.
2. Android Studio will automatically detect the SDK location and update `local.properties`.
3. Sync the project with Gradle files.

## How to Run
1. Connect a physical Android device with NFC support.
2. Enable **Developer Options** and **USB Debugging** on the device.
3. Run the app from Android Studio (Shift+F10).
4. Bring an NFC tag close to the back of the device.

## Requirements
- Android SDK Platform 31 (Android 12)
- Physical Android Device with NFC
